360 全家桶将添新口味，这次是挪威的 Opera

--SHARES
分享到微信
使用微信扫码将网页分享到微信
360 Chrome IE Java Opera UC 昆仑万维 浏览器

喜欢折腾浏览器的网民不会对 Opera 浏览器陌生，这家来自北欧的浏览器曾经有过显赫的历史。即便现在因为没有背靠 Google、微软这样的大佬而沦为小众浏览器，但是 Opera 浏览器的口碑还是不错，比如当年推出的全手势操控的移动端浏览器 Coast 就让人耳目一新。

四处征战的中国互联网巨头，这一次把目标瞄准了 Opera。挪威浏览器开发商 Opera 在当地时间周二晚上证实，中国互联网公司昆仑万维和奇虎 360 提交了收购要约，对其估值为 12 亿美元。

昆仑万维和奇虎 360 对 Opera 的出价约为 104 亿挪威克朗，相比出现收购传闻前的收盘价溢价 53%。Opera 董事会一致建议股东批准此次收购。

Opera CEO 拉尔斯 · 鲍尔森（Lars Boilesen）表示：

“中国财团收购 Opera 有着很强的战略和行业逻辑性。财团的所有权将加强 Opera 服务用户及合作伙伴的地位，可提供更好的创新技术，加快我们的扩展和增长计划。”

被中国互联网巨头收购对于 Opera 来说算是一个不错的归宿，毕竟就现在的份额来看，Opera 浏览器无论是在移动端还是桌面端都趋于边缘化，去年年中的几家统计机构数据表明，Opera 浏览器在桌面端的份额仅仅不到 2%，远远落后于 IE、Chrome、Firefox 和 Safari。

作为比较，当初阿里巴巴收购 UC 的价格据称是 43.5 亿美元。

另外一个问题是，为什么昆仑万维这家手游厂商参与了这场收购？从目前的一些信息来看，360 和昆仑万维的关系是千丝万缕，比如在 Google Play 上架的 360 Security 不是 360 嫡系产品，而是由一家摩比神奇的开发商开发的，摩比神奇则是由奇虎 360 和昆仑万维两家上市公司合资的一家新的互联网公司。

而 360 在手游方面一直都是和昆仑万维来合作的，大致的猜测是，在涉及到海外战略，以及游戏业务的时候，360 和昆仑万维的合作可能性就非常之大。

360 Security 对标的是猎豹，而 360＋昆仑万维＋Opera 的组合模型就可能类似于当初 UC＋九游。

只是，以后不知道会不会出现安装 Opera 送 360 全家桶的情况，反正百度和甲骨文合作之后，装 Java 中文版开始送百度全家桶了。
